# HELIUM MOUSEEFFECT

Projet interactif réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/Helium-MouseEffect/)

## Objectif

Créer un curseur personnalisé en JavaScript natif : trois cercles concentriques suivent la souris avec des vitesses de transition différentes et un effet mix-blend-mode pour interagir visuellement avec le contenu de la page.

## Compétences travaillées

- Manipulation du DOM avec JavaScript natif
- Création dynamique d'éléments via JavaScript
- Gestion des événements (mousemove)
- Positionnement CSS avancé et effets visuels (mix-blend-mode)

## Stack technique

JavaScript natif. HTML natif. CSS natif. Aucun framework ou bibliothèque externe.
